# 阶段 3 — 模板设计

> 当前阶段：3 | 上一阶段：2 | 下一阶段：4 | 可跳过：是（使用默认模板）

## 本阶段目标

匹配学科模板 → 个性化调整 → 用户确认 → 敲定模板。完成后进入阶段4。

## 前置条件

- 档案已保存至 `data/profiles/`
- 已知用户目标岗位（来自阶段1，如有）

## 执行步骤

### 3.1 学科模板匹配

根据用户目标岗位 + 档案数据中的技能和经历，自动匹配最佳学科模板。
**如果阶段1已录入岗位信息，JD中的行业/技术关键词会提高匹配精度。**

列出匹配结果：

```
📐 模板设计

根据您的目标岗位「<公司> · <岗位>」（来自阶段1），
匹配学科模板：**<学科名>**

该学科最优简历格式：
┌────────────────────────────────────┐
│  Section 顺序                       │
│  1. <section1>（<侧重说明>）        │
│  2. <section2>（<侧重说明>）        │
│  3. <section3>（<侧重说明>）        │
│  ...                                │
│                                     │
│  设计风格：<主题名>                  │
│  技能展示：<进度条/标签>             │
│  篇幅目标：<1-2页>                   │
│  设计理念：<该学科核心设计原则>       │
└────────────────────────────────────┘
```

可用的 16 个学科模板：
```
计算机科学 · 电子信息 · 金融经济 · 会计审计 · 工商管理
市场营销 · 法学 · 教育师范 · 医学药学 · 机械制造
土木建筑 · 设计艺术 · 新闻传媒 · 外语翻译 · 生化环材
数理统计 · 通用
```

### 3.2 个性化调整

在学科模板基础上，根据用户具体情况调整。
**如果阶段1有岗位信息，JD要求在调整时有更高权重**：

```
🔧 个性化调整建议：

基于您的具体情况：
· <用户特征1> → <调整建议1>
· <用户特征2> → <调整建议2>

📌 岗位导向（来自阶段1）：
· JD要求"<关键词>" → Section权重提升
· 行业特点 → <布局偏好>

调整后的 Section 计划：
  1. <section1> ✓ 学科默认
  2. <section2> 🔧 <调整说明>
  3. <section3> ✓ 保留
  ...

请确认或提出修改。
```

用户可以：
- 「确认」→ 进入 3.3 敲定
- 「增加XX板块」→ 更新计划后重新展示
- 「移除XX板块」→ 更新计划后重新展示
- 「调整顺序」→ 更新计划后重新展示
- 「换一个学科模板」→ 回到 3.1

### 3.3 敲定模板

用户确认后，模板锁定：

```
✅ 模板已锁定：<学科名> / <主题名>
   Section顺序：<A> → <B> → <C> → <D>

接下来将逐段填充内容，每段提供3种润色方案供您选择。
```

### 3.4 完成

输出：「进入阶段4 — 逐段内容填充」
