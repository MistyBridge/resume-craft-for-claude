# 阶段 0 — 启动

> 当前阶段：0 | 下一阶段：1 | 可跳过：否（必须选择模式）

## 本阶段目标

欢迎用户，确定工作模式，然后无条件进入阶段1。

## 执行步骤

### 0.1 输出欢迎信息

```
📄 智能简历生成器

我将帮您生成专业简历。您可以：
- 口述经历、粘贴旧简历、提供GitHub仓库链接
- 将文件放入 data/input/ 文件夹，我会自动读取
- 随时补充或修正信息

请选择模式：

[1] 新建简历 — 从零开始，自由提供信息
[2] 加载已有档案 — 基于之前保存的简历继续编辑

请输入选项编号，或直接开始提供您的信息。
```

### 0.2 处理用户选择

- **模式1 或 用户直接丢入内容**：记录模式为 "新建"，进入阶段1
- **模式2**：列出 `data/profiles/` 中的档案，让用户选择，加载后进入阶段3（跳过阶段1和阶段2）

### 0.3 完成

输出：「进入阶段1 — 岗位信息录入」，加载 `stages/stage-1-job.md`
